# KBT Staff Contacts

This repository stores vCard (`.vcf`) files for KBT staff contacts.

## Files

- `junno_contact_with_photo.vcf`: vCard file for **Junno Yau**, including contact details and embedded photo.

## Usage

1. Click on the `.vcf` file in GitHub.
2. Click the **Download** button (or "View raw") to download the file.
3. Open it on your phone or computer to add the contact to your address book.
